=====================
Команди

1. git init
    Результат появиться папка .git
2. git add .
    Додавання всіх змінених файлів 
3. git commit -m "Init commit, my first commit SN"
    Створення коміта
4. git remote add origin https://bitbucket.sense.bank.int/scm/~nskorobagatyi/test-git.git
    Связуємо локальний репозиторій з віддаленим
5. git push -u origin master
    Додаємо зміни (коміти) на віддалений репозиторій
========================================================================

(!) Команди з урока
1. git status
    Допомагає побачити список змінених файлів

2. dit diff
    Які були зміни в усіх змінених файлах
2.1 git diff css/main.css - зміни в конкретному файлі

(!) Git - працює зі строками і він не розуміє зміну в одному символі (навіть якщо вставлений пробіл), 
        якщо змінився тільки 1 символ, то Git розуміє що 1 строка була видалена, а інша вставлена

3. git restore (file name)
    ex. git restore e.html - повернення файлу в первичний вигляд
