==============================================
GIT
==============================================
Git — розподілена система керування версіями файлів 

В Git файли можуть знаходитися в одному із 3-х станів: 

- 1. зміненому (файл було змінено, але зміни не зафіксовано) 
    Була проведена тіль зміна в файлі
- 2. підготовленому (файли було змінено і відмічено для фіксації).
    Була зміна в файлі і вони підготовлені для коміта
- 3. зафіксованому (файл вже збережено в локальній базі даних), 
    (!) Файл вже закомічений

Commands:
1. git --version
2. git config --list
    Список настройок (конфігурації)
3.  git init --help open page in browser
    git add --help  from console to browser (!)

4.  clear (Ctrl+l(L) ) - clear console (cls - in command prompt)
    ls (dir) - list folders and files
